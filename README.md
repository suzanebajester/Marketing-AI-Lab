# Campaign Generator AI 🚀

AI-powered marketing campaign generator built with Lovable, Typebot, and OpenAI.

Generate complete marketing campaigns in minutes by answering a few simple questions.

## 🌐 Live Demo

https://campaign-generator-ai.lovable.app/

## 📋 Features

The application generates:

- Buyer Persona
- Campaign Headline
- Primary CTA
- Social Media Post
- Email Copy
- Creative Suggestions

All outputs are tailored to the user's:

- Industry / Niche
- Target Audience
- Product or Service
- Campaign Goal
- Language
- Tone of Voice

## 🛠️ Tech Stack

### Frontend
- Lovable

### Conversational Interface
- Typebot

### AI
- OpenAI API

## ⚙️ How It Works

1. User opens the website.
2. Clicks **Generate Campaign**.
3. Completes a short marketing brief.
4. Typebot collects the information.
5. OpenAI generates a complete campaign.
6. Results are displayed instantly.

## Example Workflow

```text
Industry → Audience → Offer → Goal → Language → Tone
                      ↓
                    OpenAI
                      ↓
        Complete Marketing Campaign
```
